## Getting Started

# prisma & planet scale

```
> npm i prisma -D
> npx prisma init
프리즈마 폴더가 생기고 .env 파일이 생성됨

> npx prisma db push
> npx prisma generate

> npm install @prisma/client

```

First, run the development server:

```
> pscale auth login
> pscale connect mobick-public-market
> npx prisma studio
```

```
> npx prisma generate
> npx prisma db push
```

First, run the development server:

```bash
npm run dev
```
